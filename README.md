## Remaps

Custom remaps will be documented here.

### Normal Mode
- `<leader>yp` - duplicate current line (preserve cursor position)
- `<C-d>`, `<C-u>` - centered half-page jumps
- `<leader>t` - toggle line wrap (useful for Markdown and LaTeX)
- `<leader>pv` - file explorer
- `<leader>mx` - compile mode (command runner)

### Visual Mode
- `<leader>y` - yank to system clipboard
- `J`, `K` - move entire selected line
- `<`, `>` - corrected indent

### LSP
- `K` - open hover details menu
- `gd` - go to definition
- `gi` - go to implementation
- `gr` - go to references
- `g]` - next diagnostic
- `g[` - previous diagnostic
- `<leader>.` - code actions
- `<leader>f` - format document

### Blink (Autocomplete)
- `<C-n>`, `<C-p>` (autocomplete window) - cycle entries
- `<C-y>` (autocomplete window) - confirm selected autocomplete entry

### Telescope
- `<leader>pf` - find files
- `<leader>ps` - live grep
- `<leader>pc` - colorscheme picker
- `<leader>pd` - man pages
- `<leader>ph` - help pages
